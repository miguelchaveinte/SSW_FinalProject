Descripción del curso

Si estás aprendiendo a programar por primera vez, o si vienes de un lenguaje diferente, este curso, JavaScript: Primeros pasos, te dará los fundamentos para codificar en JavaScript. En primer lugar, descubrirá los tipos de aplicaciones que se pueden crear con JavaScript y las plataformas en las que se ejecutarán.

A continuación, explorará los fundamentos del lenguaje, con multitud de ejemplos. Por último, pondrás en práctica tus conocimientos de JavaScript y modificarás una página web moderna y con capacidad de respuesta. Cuando termine este curso, tendrá las habilidades y el conocimiento de JavaScript para crear programas simples, crear aplicaciones web simples y modificar páginas web.

¿Qué aprenderás?

Reconocer la importancia de comprender sus objetivos al dirigirse a un público.
Identificar los fundamentos para componer un cierre exitoso.
Explorar cómo conectar con su público mediante la elaboración de historias convincentes.
Examinar las formas de conectar con su audiencia mediante la personalización de sus contenidos.
Desglosar las mejores formas de exudar presencia ejecutiva.
Explorar cómo comunicar lo desconocido en una comunicación improvisada.
